1. 子元素 width 总和 < 父元素 width, 需要扩张

100px
flex-grow: 0
50px
flex-grow: 1
    .parent{
        width: 200px;
        display: flex;
    }
    .parent div:first-of-type{
        flex: 0 0 auto; // 不占用剩余空白
        width: 100px;
    }
    .parent div:last-of-type{
        flex: 1 0 auto; //占用剩余空白
        width: 50px;
    }

2. 子元素 width 总和 > 父元素 width, 需要收缩

100px
flex-shrink: 2
200px
flex-shrink: 0
    .parent{
        width: 200px;
        display: flex;
    }
    .parent div:first-of-type{
        flex: 1 0 auto; // 不收缩
        width: 100px;
    }
    .parent div:last-of-type{
        flex: 1 2 auto; // 总 width 超出100px, 但是搜索率为 50%
        width: 200px;
    }

3. basic 顶替了 width

100px
flex-basic: 100px
200px
flex-basic: 100px
    .parent{
        width: 200px;
        display: flex;
    }
    .parent div:first-of-type{
        flex: 1 0 100px;
        width: 100px;
    }
    .parent div:last-of-type{
        flex: 1 0 100px;
        width: 200px;   // 200px 失效
    }
